## Description

Grants the NitroBoost permission when a linked Discord user is actively boosting the guild (premium_since) or has a Booster role. Hard-fails unless Image Library + Rust Kits + Custom Auto Kits are installed and Discord is configured.

## Permissions

| Permission | Description |
| --- | --- |
| NitroBoost | Granted automatically to players who qualify via Nitro boost or Booster role. Use this permission in Custom Auto Kits to unlock VIP kits. |

## Commands

| Command | Arguments | Description |
| --- | --- | --- |
| `nitrolink` | `[DiscordUserID|help]` | Start the Discord link flow (or show help). Available to all players. |
| `nitroverify` | `<CODE>` | Complete Discord link verification using the code DM'd by the bot. |
| `nitrostatus` | `—` | Show the player's current link/boost status. |
| `nitroresync` | `[player]` | Admin: re-check boost/role status for one player or all linked players. |
| `nitrodiscordbotlink` | `<BotToken> <GuildId> [BoosterRoleId|RoleName]` | Admin/console/RCON: configure Discord bot credentials at runtime. |
| `nitrodiag` | `—` | Admin/console/RCON: print dependency/config health and write to oxide/logs/NitroBoostLinker.txt. |

## Installation

1. Download `NitroBoostLinker.cs` from the latest release.
2. Upload it to `oxide/plugins/NitroBoostLinker.cs` on your server.
3. The plugin compiles and loads automatically; a default config is written to `oxide/config/NitroBoostLinker.json` on first load.
4. Install the required dependencies listed above first.
